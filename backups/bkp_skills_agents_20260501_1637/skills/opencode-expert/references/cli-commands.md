# OpenCode CLI Command Reference

## Session Management
- `opencode`: Starts an interactive session.
- `opencode --model <name>`: Starts a session with a specific model.
- `opencode --agent <path>`: Starts a session using a specific agent definition.

## Tooling
- `opencode mcp`: Manage MCP server integrations.
- `opencode skill`: Manage local and global skills.

## Configuration
- `opencode config`: View or modify the global configuration.
- `opencode config set <key> <value>`: Set a specific configuration value.
