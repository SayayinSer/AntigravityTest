# Mercury Design System (Nexa Integration)

Guía para la creación de interfaces de usuario premium utilizando el sistema de diseño oficial de **GeneXus**.

## 1. Diseño Atómico (Atomic Design)
En Mercury, se prioriza la construcción de componentes reutilizables:
- **Átomos:** Botones, Inputs, TextBlocks.
- **Moléculas:** Cards con datos, Formularios simples.
- **Organismos:** Headers, Footers, Sidemenu.

## 2. Flex y Grid Layouts
Diseño responsivo nativo:
- **Flex:** Uso de alineación horizontal y vertical fluida.
- **Grid:** Estructura de columnas para dashboards y listados complejos.

## 3. Tokens de Diseño
No uses valores "hardcoded". Usa tokens para mantener la coherencia:
- **Color:** `Primary`, `Secondary`, `Success`, `Error`.
- **Typo:** `Title`, `Subtitle`, `Body`, `Caption`.
- **Space:** `XS`, `S`, `M`, `L`, `XL`.

## 4. Patrones de UI en GX18/Next
- **Listados (Grid/List Panel):** Aplicar estilos Mercury para mejorar la legibilidad.
- **Detalle (View Panel):** Agrupación lógica de datos con `Sections`.
- **Formularios (Edit Panel):** Validación en tiempo real y feedback visual claro.

---
*Mercury asegura que la experiencia del usuario sea consistente en todas las plataformas.*
