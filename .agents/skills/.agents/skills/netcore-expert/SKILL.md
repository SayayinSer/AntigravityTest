# .NET Core Expert - Guía de Desarrollo y Despliegue

Esta Skill está diseñada para guiar el desarrollo de aplicaciones robustas utilizando el ecosistema moderno de .NET (Core 3.1, 5, 6, 7, 8+).

## Áreas de Especialización
- **Arquitectura**: Implementación de Clean Architecture, DDD (Domain-Driven Design) y patrones de diseño (Repository, Unit of Work, CQRS).
- **Backend**: Desarrollo de Web APIs de alto rendimiento con Middleware personalizado, Filtros de Acción y Validación (FluentValidation).
- **Acceso a Datos**: Uso avanzado de EF Core, Migraciones, Optimización de Consultas y Dapper para casos de alta velocidad.
- **Publicación**: Configuración de `dotnet publish`, Perfiles de publicación (Self-contained vs Framework-dependent) y contenedores Docker.
- **Seguridad**: JWT, OAuth2, Identity Server y protección de secretos.

## Checklist de Calidad
- [ ] ¿Se utiliza inyección de dependencias correctamente?
- [ ] ¿Están los secretos fuera del código (appsettings.json vs UserSecrets/KeyVault)?
- [ ] ¿Se manejan las excepciones de forma global (Middleware)?
- [ ] ¿La API está documentada con Swagger/OpenAPI?
- [ ] ¿Los logs incluyen contexto útil (Serilog)?
